# HopBot

Bot for my Discord community server.

# Dependancies
Requies Node.js 6.0 and over, and Discord.js

# Config

config.js is hidden for security purposes.

Basically, it is just

```js
//config.js
module.exports = {
    getToken: function() {
        return 'BOT TOKEN HERE';
    }
}
```

# Other info

Run using

`node bot.js`
